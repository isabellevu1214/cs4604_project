# Project Introduction

Our product, ClearPath Finance, is a database backed budgeting system. This system is designed for the university community, including undergraduate and graduate students, new graduates, and faculty/staff. This system allows users to log and track various expenses and user income, define their spending categories, set budget limits, and monitor their overall spending and progress towards their set goals. This system also supports recurring transactions, and provides an automated notification system to alert users when they may be approaching budgeting limits. Using a normalized relational database, this system is able to maintain different types of financial information and data that is tied to each individual user. This ensures that the users budgets, goals, and spending histories are accurate across the system.
